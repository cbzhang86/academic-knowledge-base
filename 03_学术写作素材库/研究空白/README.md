## 如何利用研究空白

每条空白标注了来源精读报告（可点击追溯），按方法/理论/数据三类分节。

**选题操作**：
1. 打开某方向的 `研究空白.md`
2. 关注 **方法层面** — 哪些方法没人用过？
3. 关注 **理论层面** — 哪些解释不充分？
4. 追溯回原报告 → 看原文的「不足与展望」完整语境
5. 多个空白交叉点 = 选题突破口

## 沉淀体系总览

```
论文精读/          ← 108篇精读（原始分析）
   │
   ▼ 自动提取

元数据库/*_研究空白.md ← 不足与展望的跨篇聚合

```

**更新频率**：精读报告完成一篇后，手动或自动重新生成对应的研究空白文件即可更新。
