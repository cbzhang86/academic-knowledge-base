# 青少年研究 理论框架素材

> 从 2 篇精读报告中提取

## [[后疫情时代我国青少年的网络行为问题风险及其应对策略_精读报告|后疫情时代我国青少年的网络行为问题风险及其应对策略]]

以**生态系统理论（Bronfenbrenner）** 为核心，从六个层面构建应对框架：
1. **个体系统**——自我控制/心理弹性/情绪调节
2. **微观系统**——家庭氛围/学校环境/同伴网络
3. **中间系统**——家庭-学校-同伴的互动关系
4. **外部系统**——社区资源/网络平台/社会服务
5. **宏观系统**——文化观念/政策法规/社会规范
6. **时序系统**——后疫情时代的特殊时间窗口（复学适应/创伤后成长/发展转折点）

## [[青少年网络适应的拓扑结构分析_精读报告|青少年网络适应的拓扑结构分析：基于纵向追踪数据]]

以**网络适应动态过程模型**（王伟军等，2021）为核心，构建"适应准备—适应过程—适应持续"三阶段框架：
1. **适应准备阶段**——网络好奇心（"双刃剑"：过高或过低均不利）、网络自我效能（"总舵手"）
2. **适应过程阶段**——网络自我控制、网络信息保护、网络积极应对
3. **适应持续阶段**——网络信息搜索（"落脚点"之一）、网络学习能力（"落脚点"之二）、网络人际交往
采用**网络分析方法**（偏相关网络/EBIC-glasso/交叉滞后网络）取代传统的潜变量分析，直接观测变量间的拓扑关系。
