# 多维排斥风险框架（老龄化研究方向）

## 来源文献
- 蒲新微、余柳君（2026），《养老服务数智化转型中的排斥风险及其破解之道》，《理论探索》2026年第1期

## 框架结构

```
数智技术嵌入养老服务
        ↓
   四维排斥风险
   ├── 意愿排斥
   ├── 机会排斥  
   ├── 能力排斥
   └── 权益排斥
        ↓
   五大排斥根源
   ├── 价值张力
   ├── 资源约束
   ├── 能力失配
   ├── 机制不适
   └── 制度空档
        ↓
   四条治理策略
   ├── 价值弥合
   ├── 能力弥合
   ├── 机制调适
   └── 制度完善
        ↓
   积极数智化目标
```

## 框架要旨

### 核心逻辑链
1. 数智技术嵌入养老服务具有"双刃剑"效应
2. 老年人在意愿、机会、能力、权益四个维度面临排斥
3. 排斥的根源来自价值、资源、能力、机制、制度五个层面
4. 破解之道需要对应以上根源的系统性治理策略

### 策略—根源对应关系
| 根源 | 策略 |
|------|------|
| 价值张力 | 价值弥合（技术工具理性与人文价值理性融合） |
| 资源约束 / 能力失配 | 能力弥合（数字素养教育 + 适老化产品研发） |
| 机制不适 | 机制调适（跨部门协同治理机制） |
| 制度空档 | 制度完善（法律法规、标准规范、监管体系） |

## 框架特色
1. **四维-五源-四策的三层递进**：问题描述→根源诊断→策略回应，逻辑闭环
2. **排斥概念创新**：将社会排斥理论迁移至数智化养老服务场景
3. **根源—策略对仗**：每一条策略都对应特定根源，增强逻辑严密性

## 写作素材：框架表述方式

### 问题判断
> "数智技术在牵动养老服务扩界提质的同时也催生出新的适应性风险，老年人面临意愿、机会、能力和权益的多维排斥风险。"

### 根源揭示
> "多维排斥导致数智技术难以充分转化为养老服务效能，其根源表现在价值张力、资源约束、能力失配、机制不适与制度空档等方面。"

### 策略提出
> "据此需要采取价值弥合—能力弥合—机制调适—制度完善等治理策略，增强数智化养老服务的均等性和包容性。"

## 可迁移场景
- 数字政府建设中的弱势群体排斥研究
- 公共服务数字化转型中的包容性治理
- 智慧城市建设的均等性评估框架
