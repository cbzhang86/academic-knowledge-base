# 实验法与准实验设计方法论

> 公共管理学科方法论积累 | 持续更新

---

## 一、实验法的基本原理

### 1.1 什么是实验法

实验法通过**操纵自变量**、**控制混杂变量**、**随机分配**来检验因果关系。核心逻辑：
- **处理组**：接受实验干预
- **控制组**：不接受干预
- **因果推断**：两组结果的差异归因于干预

### 1.2 实验法的内部效度优势

| 威胁 | 控制方式 |
|------|---------|
| 选择偏差 | 随机分配 |
| 历史效应 | 控制组对照 |
| 成熟效应 | 短期测量 |
| 测试效应 | 控制组对照 |
| 流失偏倚 | 意图处理分析（ITT） |

## 二、公共管理领域的实验类型

### 2.1 实验室实验

**特点**：在受控环境中进行，可严格操纵变量
**适用**：行为决策、认知过程、公共物品博弈
**局限**：外部效度有限，情境人造

### 2.2 现场实验（Field Experiment）

**特点**：在真实环境中进行，保持一定的外部效度
**适用**：政策推送效应、信息干预、行为助推
**代表**：J-PAL（阿卜杜勒·拉蒂夫·贾米尔贫困行动实验室）

### 2.3 调查实验（Survey Experiment）

**特点**：在问卷中嵌入实验操纵
**适用**：公众对政策的态度、框架效应、刻板印象
**优势**：样本量大，可远程实施

### 2.4 审计实验（Audit Study）

**特点**：派" testers" 以相同特征但不同身份接触服务提供者
**适用**：歧视研究、公共服务可及性
**代表**：简历投递实验（resume audit studies）

## 三、准实验设计

当随机分配不可行时，使用准实验设计：

### 3.1 断点回归设计 (RDD)

**原理**：在临界值附近，处理状态发生跳跃
**适用**：政策门槛效应、资格边界效应
**要求**：临界值附近样本量足够

### 3.2 双重差分法 (DID)

**原理**：比较处理组与控制组在干预前后的变化差异
**假设**：平行趋势假设（处理组与控制组在干预前趋势相同）
**代表文献**：吴明宇、郭雪松（2026）— 京津冀"协同应急"政策效应评估

**DID 扩展**：
- **交错 DID**（Staggered DID）：不同单位在不同时间接受处理
- **合成控制法**：构建反事实对照组

### 3.3 倾向得分匹配 (PSM)

**原理**：为处理组单位匹配特征相似的控制组单位
**适用**：观察性研究，减少选择偏差
**局限**：只能控制可观测的混杂因素

### 3.4 工具变量法 (IV)

**原理**：使用外生的工具变量隔离自变量的因果效应
**适用**：内生性问题（反向因果、遗漏变量）
**要求**：工具变量的有效性（相关性和排他性）

## 四、双重差分法的实施要点

### 4.1 经典 DID 步骤

1. **定义处理组与控制组**：明确哪些单位接受了干预
2. **确定干预时间点**：明确政策实施的时间
3. **构建 DID 面板数据**：干预前后多期数据最佳
4. **检验平行趋势假设**：干预前的处理组与控制组趋势应相同
5. **估计 DID 模型**：
   - 双向固定效应：Y_it = α + β(DID) + γ_i + δ_t + ε_it
   - 或加入控制变量：Y_it = α + β(DID) + X_it'γ + γ_i + δ_t + ε_it

### 4.2 平行趋势检验

**干预前趋势**：处理组与控制组应无显著差异
**检验方法**：
- 事件研究法（Event Study）：绘制干预前后各期的系数置信区间
- 干预前期的系数应不显著异于0

### 4.3 DID 的稳健性检验

- 安慰剂检验：假设假的干预时间或处理组
- 不同控制组：排除特定控制组是否影响结果
- 不同时间窗口：改变样本时间范围
- 排除其他政策干扰

## 五、代表文献解析

### 吴明宇、郭雪松（2026）— 跨域协同应急

**方法论要点**：
- **研究设计**：DID，2011-2023年中国省际面板数据
- **处理变量**：2017年京津冀"协同应急"政策
- **结果变量**：生产安全事故死亡人数、亿元产值死亡率
- **控制组**：其他省份（非京津冀）
- **固定效应**：省份固定效应 + 年份固定效应
- **标准误**：聚类到省级层面

**代码示例（Stata）**：
```stata
gen treat = (province == "北京" | province == "天津" | province == "河北")
gen post = (year >= 2017)
gen did = treat * post

reg death_rate did i.year i.province, vce(cluster province)
```

## 六、实验法的报告规范

### CONSORT 声明要点

1. **随机化方法**：如何生成随机分配序列
2. **分配隐藏**：研究者是否知道分配结果
3. **盲法**：是否对受试者/研究者盲
4. **样本量计算**：事前功效分析
5. **主要和次要结局**：预先设定

### 预注册

- 在 OSF、AEA RCT Registry 等平台预注册实验方案
- 明确主要假设和分析计划
- 避免 p-hacking 和发表偏倚

---

## 七、局限性

1. **外部效度有限**：特定情境的实验结论不一定可推广
2. **人为性**：实验室环境可能改变真实行为
3. **伦理约束**：公共政策难以随机分配个人
4. **干预强度**：政策干预难以完全标准化
